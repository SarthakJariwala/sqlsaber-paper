- **`drivers`** table has `nationality` to filter for Japanese drivers
- **`results`** table has `time` (null = did not finish) and `driverId`
- **`races`** table has `year` to filter 2007–2009


**Executing SQL:**

```sql

SELECT 
    CAST(COUNT(CASE WHEN r.time IS NOT NULL THEN r.driverId END) AS REAL) * 100 
    / COUNT(r.driverId) AS race_completion_percentage
FROM results r
JOIN drivers d ON r.driverId = d.driverId
JOIN races ra ON r.raceId = ra.raceId
WHERE d.nationality = 'Japanese'
  AND ra.year BETWEEN 2007 AND 2009;

```
