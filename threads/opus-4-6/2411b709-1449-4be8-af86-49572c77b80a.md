The dates are in `YYYY-MM-DD` format. Now let me write and execute the query to calculate the growth rate.


**Executing SQL:**

```sql

SELECT
    SUM(CASE WHEN STRFTIME('%Y', l.date) = '1996' THEN l.amount ELSE 0 END) AS total_amount_1996,
    SUM(CASE WHEN STRFTIME('%Y', l.date) = '1997' THEN l.amount ELSE 0 END) AS total_amount_1997,
    ROUND(
        (SUM(CASE WHEN STRFTIME('%Y', l.date) = '1997' THEN l.amount ELSE 0 END) * 1.0
         - SUM(CASE WHEN STRFTIME('%Y', l.date) = '1996' THEN l.amount ELSE 0 END))
        / SUM(CASE WHEN STRFTIME('%Y', l.date) = '1996' THEN l.amount ELSE 0 END) * 100,
        2
    ) AS growth_rate_pct
FROM loan l
JOIN account a ON l.account_id = a.account_id
JOIN disp d ON a.account_id = d.account_id
JOIN client c ON d.client_id = c.client_id
WHERE c.gender = 'M'
  AND STRFTIME('%Y', l.date) IN ('1996', '1997')

```
