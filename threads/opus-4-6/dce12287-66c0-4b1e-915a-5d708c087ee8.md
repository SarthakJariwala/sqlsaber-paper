We need to join `molecule`, `atom`, and `bond` (via `connected`) to compute this.


**Executing SQL:**

```sql

SELECT 
    CAST(SUM(CASE WHEN b.bond_type = '-' THEN 1 ELSE 0 END) AS REAL) / COUNT(DISTINCT a.atom_id) AS avg_single_bonds
FROM molecule m
JOIN atom a ON a.molecule_id = m.molecule_id
JOIN connected c ON c.atom_id = a.atom_id
JOIN bond b ON b.bond_id = c.bond_id
WHERE m.label = '+'

```
