Now I have a clear picture. The `set_translations` table tracks which sets have translations in various languages, and the `sets` table has the `isNonFoilOnly` flag. Let me write the query.

**Goal:** Among all sets that have a Japanese translation, find what percentage of them are non-foil only (`isNonFoilOnly = 1`).


**Executing SQL:**

```sql

SELECT 
    CAST(SUM(CASE WHEN s.isNonFoilOnly = 1 THEN 1 ELSE 0 END) AS REAL) * 100 
    / COUNT(*) AS percentage_non_foil_only
FROM sets s
JOIN set_translations st ON s.code = st.setCode
WHERE st.language = 'Japanese';

```
