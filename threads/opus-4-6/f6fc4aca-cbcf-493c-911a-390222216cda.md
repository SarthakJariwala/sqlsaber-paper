Now I have a clear picture of the schema. The `cards` table has the `isStorySpotlight` flag, and the `foreign_data` table has the `language` column. They are linked via the `uuid` column. Let me now write and execute the query.


**Executing SQL:**

```sql

SELECT 
    CAST(COUNT(CASE WHEN fd.language = 'French' THEN fd.id END) AS REAL) * 100 
    / COUNT(fd.id) AS percentage_french
FROM cards c
JOIN foreign_data fd ON c.uuid = fd.uuid
WHERE c.isStorySpotlight = 1;

```
