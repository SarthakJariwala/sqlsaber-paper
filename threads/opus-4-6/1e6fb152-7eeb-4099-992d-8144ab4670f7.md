I need to find all atoms involved in single bonds, then calculate what percentage of those atoms are chlorine. Let me write and run the query:


**Executing SQL:**

```sql

SELECT 
    ROUND(
        CAST(SUM(CASE WHEN a.element = 'cl' THEN 1 ELSE 0 END) AS REAL) * 100 
        / COUNT(a.atom_id), 
    4) AS percentage_chlorine
FROM atom a
INNER JOIN connected c ON a.atom_id = c.atom_id
INNER JOIN bond b ON c.bond_id = b.bond_id
WHERE b.bond_type = '-';

```
