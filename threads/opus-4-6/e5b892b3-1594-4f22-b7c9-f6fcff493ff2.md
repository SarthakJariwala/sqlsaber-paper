**Executing SQL:**

```sql

SELECT 
    CAST(SUM(CASE WHEN a.alignment = 'Good' THEN 1 ELSE 0 END) AS REAL) * 100.0 / COUNT(*) AS percentage_good
FROM superhero s
INNER JOIN publisher p ON s.publisher_id = p.id
INNER JOIN alignment a ON s.alignment_id = a.id
WHERE p.publisher_name = 'Marvel Comics';

```
