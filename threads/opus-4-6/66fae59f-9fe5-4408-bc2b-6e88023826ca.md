- **`Player`** table has the `birthday` column
- **`Player_Attributes`** table has `sprint_speed` and `date` columns
- They join on `player_api_id`


**Executing SQL:**

```sql

SELECT DISTINCT
    p.player_name,
    p.birthday,
    CAST((julianday('now') - julianday(p.birthday)) / 365.25 AS INTEGER) AS current_age
FROM Player p
INNER JOIN Player_Attributes pa
    ON p.player_api_id = pa.player_api_id
WHERE pa.sprint_speed >= 97
  AND CAST(strftime('%Y', pa.date) AS INTEGER) >= 2013
  AND CAST(strftime('%Y', pa.date) AS INTEGER) <= 2015
ORDER BY current_age;

```
