Let me write and execute the query:


**Executing SQL:**

```sql

SELECT DISTINCT T.team_long_name
FROM Team T
JOIN Team_Attributes TA ON T.team_api_id = TA.team_api_id
WHERE strftime('%Y', TA.date) = '2012'
  AND TA.buildUpPlayPassing IS NOT NULL
  AND TA.buildUpPlayPassing > (
      SELECT CAST(SUM(buildUpPlayPassing) AS REAL) / COUNT(team_api_id)
      FROM Team_Attributes
      WHERE strftime('%Y', date) = '2012'
        AND buildUpPlayPassing IS NOT NULL
  )
ORDER BY T.team_long_name;

```
