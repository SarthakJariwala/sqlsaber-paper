**Executing SQL:**

```sql

SELECT 
    (CAST(SUM(CASE WHEN STRFTIME('%Y', Date) = '2010' THEN 1 ELSE 0 END) AS REAL) / COUNT(Name)) * 100 
    - 
    (CAST(SUM(CASE WHEN STRFTIME('%Y', Date) = '2011' THEN 1 ELSE 0 END) AS REAL) / COUNT(Name)) * 100 
    AS percentage_difference
FROM badges
WHERE Name = 'Student';

```
