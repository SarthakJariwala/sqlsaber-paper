Now I have a clear picture. Let me run the query to calculate the percentage.

- **`set_translations`** has `language` and `setCode` — this tells us which sets have a Brazilian Portuguese translation.
- **`sets`** has `isOnlineOnly` — this tells us if a set is only available online.
- We join them on `setCode = code`, filter for `language = 'Portuguese (Brazil)'`, and calculate the percentage that are online-only.


**Executing SQL:**

```sql

SELECT 
    CAST(SUM(CASE WHEN s.isOnlineOnly = 1 THEN 1 ELSE 0 END) AS REAL) * 100 
    / COUNT(*) AS percentage_online_only
FROM set_translations st
JOIN sets s ON st.setCode = s.code
WHERE st.language = 'Portuguese (Brazil)';

```
