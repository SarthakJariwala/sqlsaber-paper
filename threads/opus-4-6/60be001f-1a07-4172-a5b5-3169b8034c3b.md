**Assistant:**

Now I have a clear picture of the schema. The `cards` table has `setCode` which links to the `sets` table's `code`. Let me write the query to calculate the percentage.


**Executing SQL:**

```sql

SELECT 
  ROUND(
    CAST(SUM(CASE WHEN c.cardKingdomFoilId IS NOT NULL AND c.cardKingdomId IS NOT NULL THEN 1 ELSE 0 END) AS REAL) 
    / COUNT(*) * 100, 
  2) AS percentage_incredibly_powerful
FROM cards c
JOIN sets s ON c.setCode = s.code
WHERE s.name = 'Coldsnap';

```
