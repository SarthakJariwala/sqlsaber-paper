Here is the SQL query that calculates the total view count for posts by both users and returns the one with the higher popularity:

```sql
SELECT u.DisplayName
FROM users u
JOIN posts p ON u.Id = p.OwnerUserId
WHERE u.DisplayName IN ('Harvey Motulsky', 'Noah Snyder')
GROUP BY u.DisplayName
ORDER BY SUM(p.ViewCount) DESC
LIMIT 1;
```

**Results:**
The query returns **Harvey Motulsky**.

**Explanation:**
The query joins the users and posts tables to match each post to its author. It filters the authors to look only for Harvey Motulsky and Noah Snyder, and then adds up the view counts for all of their respective posts. Finally, it sorts the 
users by this total view count from highest to lowest and returns the top result.
