Here is the SQL query that calculates the annual average number of sets released between 2012 and 2015, alongside identifying the overall most common card language:

```sql
SELECT
  (SELECT AVG(year_counts) 
   FROM (
       SELECT COUNT(id) AS year_counts 
       FROM sets 
       WHERE releaseDate BETWEEN '2012-01-01' AND '2015-12-31'
```
