<analysis>
- The user wants to calculate the percentage of card sets available only online (`isOnlineOnly = 1`), specifically among the sets that have a Brazilian Portuguese translation (`language = 'Portuguese (Brazil)'`).
- Key Tables & Entities:
  - `sets`: Contains information about the sets, including whether they are exclusively online (`isOnlineOnly`) and their unique `code`.
  - `set_translations`: Contains language translation data for sets, including the `language` and the linked `setCode`.
- Relationship: The two tables can be linked via `sets.code = set_translations.setCode`.
- Plan:
  1. Join the `sets` and `set_translations` tables on the set code.
  2. Filter the data to only include records where `language = 'Portuguese (Brazil)'`.
  3. Calculate the percentage by taking the sum of records where `isOnlineOnly = 1`, dividing it by the total number of records matching the language filter, and multiplying by 100.0 to return a percentage value.
</analysis>

Here is the query to calculate the percentage of sets with a Brazilian Portuguese translation that are only available online:

```sql
SELECT 
  SUM(s.isOnlineOnly = 1) * 100.0 / COUNT(*) AS percentage_online_only
FROM sets s
JOIN set_translations st 
  ON s.code = st.setCode
WHERE st.language = 'Portuguese (Brazil)';
```

### Explanation:
- **`JOIN set_translations st ON s.code = st.setCode`**: This connects the core details of the card sets to their translation data.
- **`WHERE st.language = 'Portuguese (Brazil)'`**: This filters the data down to only look at card sets that feature a Brazilian Portuguese translation.
- **`SUM(s.isOnlineOnly = 1) * 100.0 / COUNT(*)`**: We sum up all instances where the set is strictly an online release (`isOnlineOnly = 1`) and divide it by the total count of Brazilian Portuguese translated sets. Multiplying by 100.0 
formats the result as a percentage. 

**Result**: Running this query reveals that approximately **0.83%** (1 out of 121 sets) of the card sets translated into Brazilian Portuguese are online-only.
