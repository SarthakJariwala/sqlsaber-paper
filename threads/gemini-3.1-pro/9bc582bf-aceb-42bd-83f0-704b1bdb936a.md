To answer your question, I will look at the `schools` table to find non-chartered schools in Los Angeles County, and join it with the `frpm` (Free and Reduced-Price Meals) table to calculate the percentage of eligible free meals for grades

Here is the SQL query to get this information:

```sql
SELECT
  COUNT(T1.School)
FROM schools AS T1
INNER JOIN frpm AS T2
  ON T1.CDSCode = T2.CDSCode
WHERE
  T1.Charter = 0
  AND T1.County = 'Los Angeles'
  AND T2.`Free Meal Count (K-12)` * 100.0 / T2.`Enrollment (K-12)` < 0.18;
```

### Explanation of the Query:
1. **`INNER JOIN frpm AS T2 ON T1.CDSCode = T2.CDSCode`**: This connects the school's general information with its free meal program data using the unique school code (`CDSCode`).
2. **`WHERE T1.Charter = 0 AND T1.County = 'Los Angeles'`**: This filters the results to only include traditional (non-chartered) schools located in Los Angeles County.
3. **`AND T2.Free Meal Count (K-12) * 100.0 / T2.Enrollment (K-12) < 0.18`**: This applies the formula to calculate the percentage of students eligible for free meals and strictly keeps those below 0.18%.
4. **`COUNT(T1.School)`**: Finally, it counts the total number of schools that meet all these conditions.
