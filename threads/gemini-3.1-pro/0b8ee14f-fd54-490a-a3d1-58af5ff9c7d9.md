Here is the query used to find the answer:

```sql
SELECT 
  (SUM(CASE WHEN Date LIKE '2012%' THEN Consumption ELSE 0 END) - 
   SUM(CASE WHEN Date LIKE '2013%' THEN Consumption ELSE 0 END)) / 
   SUM(CASE WHEN Date LIKE '2012%' THEN Consumption ELSE 0 END) AS decrease_rate
FROM yearmonth 
WHERE CustomerID = (
  SELECT CustomerID 
  FROM transactions_1k 
  WHERE Price = 634.8 AND Date = '2012-08-25'
);
```
