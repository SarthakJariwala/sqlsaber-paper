To find the percentage of "good" superheroes in Marvel Comics, I joined the `superhero` table with the `publisher` and `alignment` tables. I filtered for characters published by "Marvel Comics" and then calculated the proportion of those

Here is the SQL query used to calculate this:

```sql
SELECT 
  CAST(SUM(CASE WHEN a.alignment = 'Good' THEN 1 ELSE 0 END) AS REAL) * 100.0 / COUNT(*) AS good_percentage

JOIN publisher p ON s.publisher_id = p.id

WHERE p.publisher_name = 'Marvel Comics';
```
