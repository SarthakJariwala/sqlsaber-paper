Here is the SQL query to get the names and DOC types of the schools matching your criteria:

```sql
SELECT
  T1.School,
  T1.DOCType
FROM schools AS T1
JOIN frpm AS T2
  ON T1.CDSCode = T2.CDSCode
WHERE T2.`Enrollment (K-12)` - T2.`Enrollment (Ages 5-17)` > (
  SELECT AVG(T4.`Enrollment (K-12)` - T4.`Enrollment (Ages 5-17)`)
  FROM schools AS T3
  JOIN frpm AS T4
    ON T3.CDSCode = T4.CDSCode
  WHERE T3.FundingType = 'Locally funded'
);
```

### Explanation of the Query
1. **The Subquery (Bottom part)**: It looks specifically at schools where the funding model is set to `'Locally funded'`. It calculates the difference between `Enrollment (K-12)` and `Enrollment (Ages 5-17)` for each of these schools and 
computes the overall average (which happens to be roughly 16.7 students).
2. **The Main Query (Top part)**: It looks at the enrollment counts for **all** schools by joining the `schools` and `frpm` enrollment tables together using their unique identifier code (`CDSCode`).
3. **The Filter (`WHERE`)**: It compares each school's computed enrollment difference against the average found in step 1. If a school's difference is greater than the locally funded average, it outputs the name of that school (`School`) 
and its DOC type (`DOCType`).
