Here is the SQL query used to gather your requested list. I used `GROUP_CONCAT` to combine the different formats into a single, clean list for each card, making it much easier to read!

```sql
SELECT c.name, GROUP_CONCAT(DISTINCT l.format) AS legal_formats
FROM sets s
JOIN cards c ON s.code = c.setCode
JOIN legalities l ON c.uuid = l.uuid
WHERE s.name = 'Hour of Devastation'
  AND l.status = 'Legal'
GROUP BY c.name
ORDER BY c.name;
```
