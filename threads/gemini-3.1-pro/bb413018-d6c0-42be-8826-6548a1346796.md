Based on your request and the context provided, here is how we can find the elements of atoms that aren't bonded to anything:

### Database Query
We'll select the `element` from the `atom` table and filter out any atoms that have an entry in the `connected` table (which tracks bonds between atoms).

```sql
SELECT element 
FROM atom 
WHERE atom_id NOT IN (SELECT atom_id FROM connected);
```

### Explanation of Results
This query looks at every atom in the database and checks if its unique ID is missing from the table that logs atomic connections (`connected`). If it is missing, it means the atom is completely unbonded. The query then returns the chemical
element for each of these isolated atoms.
